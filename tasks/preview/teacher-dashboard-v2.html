Decisions to validate against this draft

  • Top nav, 5 items + user menu. No sidebar. Items: Hjem · Kurs · Påmeldinger · Kalender · Studio. Lokasjoner / Betalinger / Profil collapse into the user menu.
  • Page header is title + actions. No greeting, no date. SpareBank-style restraint.
  • 2×2 grid, equal-weight, gap-6 (24px). Top row primary (Kommende kurs · Aktivitet), bottom row secondary (Trenger oppmerksomhet · Inntekter).
  • Cards: p-7 (28px), white bg, light border, no inset shadow. Heads-up: p-7 isn't in CLAUDE.md's allowed list — if this looks right we add it as a sanctioned dashboard tier.
  • Page bg: very subtle warm off-white (oklch(0.985 0.002 80)). White cards float on it. SpareBank's calm-but-not-stark feel.
  • Row: text-base title (16px), text-sm meta (14px), 14px vertical padding. Two text tiers, never three.
  • Exception rows tinted with warning/4 background + circle ! icon + chip-style "Betaling …" in the meta line. Action button on the right.
  • KPI card right-aligns its link to bottom (margin-top: auto) so the card height matches the exception list.
  • No illustration, no upsell cards, no greeting. Calmer than SpareBank by virtue of being smaller-scope.
  • Activity icon: + in a muted circle. Generic for now. Could be replaced with a calendar/payment/cancel icon variants later.

Push back on any of:

  • The + circle icon in activity — generic and might get noisy. Alternative: no icon at all, just text rows like the Kommende-kurs card.
  • Bottom-right card title "Inntekter" with denne måneden as a sub-title — vs combining as one heading line.
  • Action button text colours — currently primary CTA is solid foreground (very dark), secondary is white-with-border. Inverts the dashboard's usual primary-blue. Might want to lock primary as foreground across the dashboard sweep.
  • Page max-width 1240px — feels right for 2×2 cards. Wider goes ugly fast.
  • Top nav active state uses a quiet grey pill, not a colored underline. Quieter than current dashboard's hover-blue treatment.
